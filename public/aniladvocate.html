Project Details

This is a dynamic website. Which have 3 user logins. They are clinet, Sub Advocate and Main Advocate. The clinet have the faciliy to book their Advocate on online and if the Advocate accept the request, the clinet will able to pay Advocate fee via online and Book him. The main Advocate can able add, delete and modify existing sub advocates and able to export data in excel, JPG and PNG formats

Main functions of Main Advocate:
* Basic Login, Logout and password Change facility.
* Can able to View, Edit, Delete the Details of Sub Adv.
* Able to view each sun adv perfomances.
* Able to export data in different formats.
* Data are stored in Online Mysql Database.
* Data analysis is also available.


Main functions of Sub Advocates:
* Accept or Decline requests from Client.
* Can able to View requests.
* Basic Login, Logout and password Change facility.
* Can view and manage the orders.
* Able to export data in different formats.
* Data are stored in Online Mysql Database.


Main functions of Client:
* Basic Login, Logout and password Change facility.
* Able to book advocates once approved Client can able to pay fee via online.
* Can view and manage the bookings.
* Once booked Client get email and Sms notification * Able to give Ratting and feedbacks.
